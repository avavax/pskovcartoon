# pskovcartoon
Одностраничный сайт Псковкартон
### Особенности
---
- адаптивная вёрстка (1140px, 960px, 720px, 540px)
- интерактивная Яндекс-карта
- препроцессор SCSS
- внедрённые шрифты
- бургер-меню
- font-awesome
---
### Внешние библиотеки
---
- Bootstrap 4
- jQuery 3.3.1
- слайдер Slick
- анимация WoW + Animate
